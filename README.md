# Memory-Pool Allocator & Hash Table

Реализация хэш-таблицы на C с кастомным аллокатором памяти (Arena Allocator) для цепочек узлов коллизий. Устраняет оверхед на системные вызовы malloc и борется с фрагментацией кучи

## Запуск

```
make && ./a
```

## Данные

Для генерации данных можно скомпилировать и запустить `helper.c`

## Тестирование

В рамках подготовки отчётов к лабороторным работам требовалось прикладывать скриншоты консоли с ручным вводом, решил автоматизировать тестирование программы,  создание наброска отчёта и создание скриншотов

Потенциальное развитие: автоматическая генерация .tex для отчёта
